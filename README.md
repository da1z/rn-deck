# rn-deck

Stack of beautiful cards that a user can slide around the screen, swiping right to &#39;like&#39; an item, or &#39;left&#39; to dislike an item. 

## Installation

```sh
npm install rn-deck
```

## Usage

```js
import RnDeck from "rn-deck";

// ...

const result = await RnDeck.multiply(3, 7);
```
